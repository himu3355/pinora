# 🤖 AI Assistant Knowledge Base

Welcome to the AI Knowledge Base! This directory contains instructions, context, and documentation specifically formatted for AI coding assistants (like Gemini, Cursor, Windsurf, Claude, etc.).

## 📂 Structure

This directory is modular and designed to be read by AI agents:

- `/knowledge-base` - Step-by-step guides, technical docs, and plugin instructions.
- `/project-context` - (Optional) Domain knowledge and architecture specific to this project.
- `/rules` - (Optional) Coding standards and strict rules for the AI to follow.

## 🔄 Reusability

This structure is designed to be highly portable. You can copy the entire `.ai` folder (or specific subdirectories like `.ai/knowledge-base/plugins/redberry-page-builder/`) to any other project, and your AI assistant will immediately understand how to implement those features! When you copy it to another project, the AI will use this documentation as context for related prompts.
